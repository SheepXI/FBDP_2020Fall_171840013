# README_K-Means

## 说明

「result」文件夹中存放以不同聚类个数与不同迭代次数运行后所得到的聚类结果，为txt格式。文件名皆为「KxTy.txt」，x代表聚类个数，y代表迭代次数。

「images」文件夹中存放由「KxTy.txt」得出的可视化聚类结果，命名为「KxTy.png」，同时该文件夹下还存放了可视化代码（python）
